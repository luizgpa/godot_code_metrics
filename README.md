# Godot Code Metrics

A lightweight tool for analyzing and visualizing code metrics in Godot Engine projects. This project aims to help developers track complexity, maintainability, and overall code quality over time.

## 🚧 Status

This project is in its early stages. Features and APIs are subject to change as development progresses.

## ✨ Features (Planned)

- Analysis of common software metrics, including:
  
  - CBO (Coupling Between Objects)
  
  - LCOM4 (Lack of Cohesion of Methods)
  
  - Additional maintainability and complexity indicators

- Visualization of code metrics over time

- Simple and lightweight setup

## 🎯 Goals

- Provide actionable insights into code quality for game developers

- Encourage maintainable and scalable code practices

- Offer an easy-to-use tool tailored for Godot projects
